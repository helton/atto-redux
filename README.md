# atto-redux [![Build Status](https://travis-ci.org/helton/atto-redux.svg?branch=master)](https://travis-ci.org/helton/atto-redux)

> A minimalist [Redux](http://redux.js.org/)-like library

## Usage

### Yarn

`$ yarn add atto-redux`

### NPM

`$ npm install atto-redux --save`
